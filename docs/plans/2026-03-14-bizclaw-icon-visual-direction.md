# BizClaw Icon Visual Direction

**Movement name:** Precision Claw

BizClaw 的图标应当像一枚经过反复打磨的桌面软件徽章：结构克制、边缘锐利、气质稳定，但内部仍然保留向前抓取的力量感。它不是卡通的“爪”，而是把 claw 的含义藏进几何节奏里，让人先感受到专业、秩序和掌控，再在细看时读到品牌个性。

空间上，图标要依赖少量高信噪比的大形体完成识别。外轮廓必须在极小尺寸下依然清晰，内部符号则通过一条深色立柱和一枚蓝色环抱体建立主次关系，像是被专家刻意保留下来的结构骨架。负空间不是空白，而是让符号呼吸、让标记显得昂贵的关键。

颜色上，主基调采用深海军蓝、冷白和高辨识度品牌蓝。蓝色需要显得克制而现代，不能轻浮；深色部分要像金属或夜色中的仪表面板，沉稳、干净、可信。所有高光都应当极轻，只为让图标显得经过精密抛光，而不是堆叠特效。

最终成品必须看起来像有人花了大量时间反复修边、推敲比例、验证缩放表现后留下的结果。它应该在托盘、窗口角标、应用切换器和 favicon 里都显得同样老练，像是由一支对品牌和软件质感都非常苛刻的团队，以顶级工艺水准完成的单体标。
